# Operating Systems Lab - 5th Semester

This repository contains implementations and practice code for Operating Systems lab exercises completed during the 5th semester, performed on Ubuntu.

## üìö Contents

### 1. **Banker's Algorithm**
Implementation of the deadlock avoidance algorithm for resource allocation.

### 2. **CPU Scheduling**
Various CPU scheduling algorithms including:
- First Come First Serve (FCFS)
- Shortest Job First (SJF)
- Round Robin (RR)
- Priority Scheduling

### 3. **Disk Scheduling Algorithms**
Implementations of disk scheduling techniques such as:
- FCFS
- SCAN
- C-SCAN
- LOOK
- C-LOOK

### 4. **Dynamic Storage Allocation**
Memory allocation strategies:
- First Fit
- Best Fit
- Worst Fit

### 5. **Shell Scripting (LAB 3 & 4)**
Shell scripts covering basic to intermediate scripting concepts and commands.

### 6. **Process Management (LAB 5)**
Programs demonstrating:
- `fork()` system call
- `wait()` system call
- Parent-child process relationships

### 7. **Page Replacement Algorithms**
Implementation of virtual memory page replacement policies:
- FIFO (First In First Out)
- LRU (Least Recently Used)
- Optimal Page Replacement

### 8. **Semaphore Problems**
Classical synchronization problems:
- Producer-Consumer Problem
- Reader-Writer Problem
- Dining Philosophers Problem
